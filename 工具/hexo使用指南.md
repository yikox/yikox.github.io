## 常规使用
### 新建
```shell
hexo new post "post name"
```
### 生成
```shell
hexo g
hexo generate
```
### 预览
```shell
hexo s
hexo server
```
### 发布
```shell
hexo d
hexo deploy
```

## 使用VScode插件
插件名称：`vscode-hexo`
使用方法：`cmd+shelf+p`进入命令行，输入hexo 选择命令
